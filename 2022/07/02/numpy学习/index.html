当a，b是可变对象时，b和a指向同一个内存空间，改变a中的值（a的内存空间地址不变），则b跟着改变（b的内存空间地址不变）;同样的，改变b中的值，则a跟着改变。

当a，b是不可变对象时，b和a指向同一个内存空间，改变a中的值（a的内存空间地址改变），这时b不跟着改变（因为b的内存空间地址不变）;同样的，改变b中的值，这时a不跟着改变。

可变对象：list dict set
不可变对象：tuple string int float bool
